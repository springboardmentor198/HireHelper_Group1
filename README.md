# 🚀 HireHelper – Full Stack Service & Task Management Platform:
HireHelper is a full-stack web application built to connect users and service providers on a single platform for efficient task and service management. It allows users to post requests, manage tasks, and track progress, while service providers can accept, manage, and complete assigned work.

# ✨ Features:
- User Registration & Login system  
- Role-based access (User / Service Provider)  
- Post, manage, and track service requests  
- Task assignment and status tracking  
- Profile management  
- Responsive and modern UI  

# 🛠️ Tech Stack:
Frontend: React.js, Vite, HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Database: MongoDB
Tools: Git, GitHub, VS Code

# 🚀 Installation & Setup:
Clone repository:
git clone https://github.com/springboardmentor198/HireHelper_Group1.git

Frontend setup:
cd Frontend
npm install
npm run dev

Backend setup:
cd Backend
npm install
npm start
